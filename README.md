# Vibin · Personal Portfolio

A clean, minimal personal portfolio website built with pure HTML, CSS, and JavaScript.

## Live Demo

[View Portfolio](#) <!-- Replace # with your hosted URL -->

## Features

- Sticky navigation with smooth scroll and active link highlight
- Typing animation for roles on the hero section
- About section with hobbies and interests
- Skills grouped by category
- Experience & certifications timeline
- Projects showcase
- Social media links with brand-color hover effects (LinkedIn, GitHub, Instagram, WhatsApp, X, Telegram)
- Contact form powered by Formsphere
- Fully responsive for mobile, tablet and desktop
- Fade-in scroll animations

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript
