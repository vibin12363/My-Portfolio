# Vibin · Personal Portfolio

A clean, minimal personal portfolio website built with pure HTML, CSS, and JavaScript.

## Live Demo

[View Portfolio](https://vibin-portfolio-website.netlify.app/) <!-- Replace # with your hosted URL -->

## Features

- Sticky navigation with smooth scroll and active link highlight
- Typing animation for roles on the hero section
- About section with hobbies and interests
- Skills grouped by category
- Experience & certifications timeline
- Projects showcase
- Social media links with brand-color hover effects (LinkedIn, GitHub, Instagram, WhatsApp, X, Telegram)
- Contact form powered by Formsphere
- Fully responsive for mobile, tablet and desktop
- Fade-in scroll animations

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript

## Project Structure

portfolio/
├── index.html       # Main structure
├── style.css        # All styles
└── script.js        # Typing animation, scroll spy, form, animations

## Setup

1. Clone the repository
```bash
   git clone https://github.com/vibin12363/portfolio.git
```
2. Add your photo as `your-image.jpg` in the root folder
3. Replace `#` links with your actual social media URLs in `index.html`
4. Set up [Formsphere](https://formsphere.app) and add your Form ID in `script.js`
5. Open `index.html` in a browser
